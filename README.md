# Pet Adoption Platform

## Project Name
Pet Adoption Platform

## Purpose
Pet Adoption Platform is a modern web application that helps users find and adopt pets easily. The platform connects pet owners, shelters, and adopters in one place. Users can browse available pets, view pet details, and submit adoption requests through a simple and responsive interface.

## Live URL
🔗 https://pet-adoption-platform-omega.vercel.app/

## Features
- 🐶 Browse pets available for adoption
- 🔍 Search and filter pets by species and breed
- 📄 Detailed pet information page
- 🔐 User authentication and protected routes
- ❤️ Responsive modern UI for mobile and desktop
- 📝 Add pets for adoption
- 🗑️ Update and delete pet listings
- 📦 MongoDB database integration

## Technologies Used
- Next.js
- React.js
- Tailwind CSS
- MongoDB
- Express.js
- Node.js
- JWT Authentication
